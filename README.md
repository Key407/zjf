# 拒绝采样法

## 1. 项目简介
本项目采用 **分段包络拒绝采样法**（Piecewise Envelope Rejection Sampling）模拟银河系盘面表面密度分布函数 \(p(r)\)。  
通过生成 **13 万个** 符合分布的随机样本，绘制样本统计直方图与理论概率密度曲线的对比图，验证算法的正确性。

## 2. 参数与公式
公式（来自 PPT）：  
\[
p(r) = A \left(\frac{r + R_{gal}}{R_c + R_{gal}}\right)^a \exp \left[-b \frac{r - R_c}{R_c + R_{gal}} \right]
\]

实际采用参数：  
- \(R_c = 8.3\) (kpc, 太阳到银心距离)  
- \(R_{gal} = 3.76\) (kpc)  
- \(A = 20.41\)  
- \(a = 9.03\)  
- \(b = 13.99\)

采样区间为 \(r \in [0, 25]\) kpc。

## 3. 算法实现细节
本程序采用 **分段常数包络 + 向量化拒绝采样**，具体步骤如下：

1. **构建分段包络**  
   - 将区间 \([0, 25]\) 等分为 200 个小区间。  
   - 在每个小区间内，取左端点、右端点以及区间中点的 \(p(r)\) 值的最大值作为该段的包络高度 \(M_i\)（近似上限）。  
   - 段内以常数高度 \(M_i\) 盖住目标函数，形成分段均匀的提议分布。

2. **计算抽样权重**  
   - 每段的包络面积 = 段宽 × 包络高度。  
   - 根据各段面积占总面积的比例，分配抽样概率，生成累积概率数组。

3. **向量化批量采样**  
   - 每次批量生成 50,000 个候选点。  
   - 先按累积概率随机选择采样段（`np.searchsorted`），再在段内均匀抽取候选位置 \(r_{\text{cand}}\)。  
   - 同时产生对应段的均匀随机数 \(u \in [0, M_{\text{sel}}]\)。  
   - 使用布尔掩码一次性接受所有满足 \(u \le p(r_{\text{cand}})\) 的样本点。  
   - 循环累加，直到收集满 130,000 个样本。
4. **归一化与绘图**  
   - 理论概率密度函数通过对 \(p(r)\) 在 \([0, 25]\) 上做梯形积分求得归一化常数 \(Z\)。  
   - 绘制样本直方图（`density=True`）与归一化后的理论曲线 \(p(r)/Z\) 对比，验证分布一致性。
