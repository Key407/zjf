# 拒绝采样法

## 1. 项目简介
这是采用 **拒绝采样法**（Rejection Sampling）模拟银河系盘**表面密度分布函数** \(p(r)\) 的数值模拟作业。
通过生成 13 万个符合分布的样本点，绘制样本统计直方图与理论概率密度曲线的对比图。

## 2. 参数与公式
公式（来自 PPT）：  
\[
p(r) = A \left(\frac{r + R_{gal}}{R_c + R_{gal}}\right)^a \exp \left[-b \frac{r - R_c}{R_c + R_{gal}} \right]
\]

实际采用参数：  
- \(R_c = 8.3\) (kpc, 太阳到银心距离)  
- \(R_{gal} = 3.76\) (kpc)  
- \(A = 20.41\)  
- \(a = 9.03\)  
- \(b = 13.99\)

采样区间为 \(r \in [0, 35]\) kpc。

## 3. 算法实现细节
本程序采用了 **向量化批量采样** 代替传统的 `while` 循环逐个采样，具体步骤：
1. 找到函数在采样区间内的最大值 \(M\)（并保留 5% 的安全余量）；
2. 每次批量生成 50,000 个候选点 \(r_{cand} \in [0, 35]\) 和对应的判定随机数 \(u \in [0, M]\)；
3. 利用 NumPy 的布尔掩码一次性筛选出所有满足 \(u \le p(r_{cand})\) 的样本；
4. 循环累加直至样本数量达到 130,000 个。

在绘图时，为了视觉清晰：
- 直方图设置了 `rwidth=0.8`，确保每列之间存在肉眼可见的空隙；
- 理论曲线除以归一化常数 \(Z\)（梯形积分求得），使其与直方图在同一概率密度尺度上对比。
